HTML
HTML (Hypertext markup language) is the set of markup symbols or codes inserted in a file intended for display on a World Wide Web browser page. The markup tells the Web browser how to display a Web page's words and images for the user. Each individua markup code is is referred to as an element (but many people refer to it as a tag). Some elements come in pairs that indicate when some display effect is to begin and when it is to end.
CSS
CSS is a stylesheet language that describes the presentation of an HTML (or XML) documents. CSS describes how elements must be render on screen, on paper, ot in other media.
